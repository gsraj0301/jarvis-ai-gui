Description:

JARVIS is a desktop-based voice-controlled AI assistant that automates various system-level tasks. It can respond to voice commands to open websites, read pdfs,and perform operations like telling time, launching apps, and more.

Current Features:
- Voice command recognition using SpeechRecognition
- Text-to-speech response via pyttsx3
- Opens websites like Google and YouTube
- Basic automation (system time, file opening)

Upcoming Features:
- A more user friendly GUI interface
- More features

Tech Stack:
- Python
- pyttsx3
- SpeechRecognition
- Webbrowser, OS, PyPDF2
- And many more
